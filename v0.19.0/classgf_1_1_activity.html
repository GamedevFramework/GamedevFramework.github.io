Detailed Description

A game activity.

An activity is a task that can be run over time. gf provides predefined activites but you can also add your own activities.

Here is a list of predefined activites:

  • gf::ValueActivity
  • gf::RotateToActivity
  • gf::MoveToActivity
  • gf::ColorActivity
  • gf::CallbackActivity
  • gf::DelayActivity
  • gf::SequenceActivity
  • gf::RepeatActivity
  • gf::ParallelActivity
