Detailed Description

Generic color matrix effect.

This effect uses a color matrix. You can use it directly or use a subclass that defines their own color matrix for various needs.

See also
gf::ColorEffect, gf::ColorBlindEffect

void gf::ColorMatrixEffect::setColorMatrix ( const Matrix4f &  mat)

Set the color matrix.

You can use this function to set your own color matrix. The color of the texture is left-multiplied by the color matrix to get the final color.
