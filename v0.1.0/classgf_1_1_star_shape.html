Detailed Description

Specialized shape representing a star.

This class inherits all the functions of gf::Transformable (position, rotation, scale, bounds, ...) as well as the functions of gf::Shape (outline, color, texture, ...).

Usage example:

gf::StarShape star;
star.setMinRadius(25.0f);
star.setMaxRadius(50.0f);
star.setBranches(5);
star.setOutlineColor(gf::Color::Red);
star.setOutlineThickness(5);
star.setPosition({ 10.0f, 20.0f });
...
window.draw(star);
